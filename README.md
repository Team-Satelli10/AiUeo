# AiUeo
好きなキャラクターと自由に交流しよう！

---
現在はMacOSおよびWindows11に対応しております。Linux環境は未検証です。

## 動作要件  
### Windows  
・CPU Intel Core i7 10th Gen~ or AMD Ryzen 7 5000 ~  
・RAM DDR4 16GB~  
・GPU Nvidia Geforce RTX2000番台~ or AMD Radeon RX5000番台   
・VRAM 8GB~  
・Storage 空き容量128GB~  

### macOS  
・CPU Apple M3 Pro~  
・RAM 16GB~  
・Storage 空き容量128GB~
