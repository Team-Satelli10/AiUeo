# AiUeo
好きなキャラクターと自由に交流しよう！
このページはまだセットアップ中です。申し訳ありません！

---
現在シングルマシンタイプはMacOS,Windows11、クラウドタイプはAndroid,macOS,windows11に対応しております。Linux,iOS,iPadOS環境は未検証です。

## 動作要件(シングルマシン)  
### Windows  
・CPU Intel Core i7 10th Gen~ or AMD Ryzen 7 5000番台~  
・RAM DDR4 16GB~  
・GPU Nvidia Geforce RTX2000番台~ or AMD Radeon RX5000番台   
・VRAM 8GB~  
・Storage 空き容量128GB~  

### macOS  
・CPU Apple M3 Pro~  
・RAM 16GB~  
・Storage 空き容量128GB~  

## 動作要件(クラウドタイプ)  
### Windows  
・CPU Intel Core i5 10th Gen~ or AMD Ryzen 5 5000番台~  
・RAM DDR4 16GB~  
・GPU Nvidia Geforce RTX2000番台~ or AMD Radeon RX5000番台   
・VRAM 8GB~  
・Storage 空き容量64GB~  

### macOS  
・CPU Apple M3 Pro~  
・RAM 16GB~  
・Storage 空き容量64GB~  

### AndroidOS  
・CPU Snapdragon 7+ Gen3またはそれ以上の性能を持つSoC  
・RAM 10GB~(仮想ストレージ使用不可)  
・Storage 空き容量32GB~
